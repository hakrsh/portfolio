I made a [Webhooks plugin for Medusa](https://github.com/AnishDe12020/medusa-plugin-webhooks) which lets one send webhooks to any URL when an event (like an order being placed) occurs in your store powered by Medusa. This plugin won the "Most Technically Impressive" award in the hackathon.
